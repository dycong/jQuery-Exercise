8:jquery 对象 与 dom 对象的理解
    jQuery 对象是 $("") 去获取页面上面的元素得到的对象。jquery 对象可以调用jQuery 对象下面的方法
    dom 对象是 document 去获取页面上面的元素得到的对象。document 对象可以调用document 对象下面的方法
    对象的转换:
     jQuery 对象是 $("") 去获取页面上面的元素得到的对象。不能调用document 对象属性以及方法
     但是可以转换成dom 对象
     $("#btn")[0]
     $("#btn").get(0)
     dom 对象是 document 去获取页面上面的元素得到的对象，不能调用JQuery 里面的属性以及方法
     但是dom 对象可以转换成jQuery 对象
     var btn = document.get.....;
     var $btn = $(btn);
